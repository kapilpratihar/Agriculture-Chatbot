In this project we use google gemini api services to make chatbot.
we can also use openai services but because of plan or billing service in openai we use this.

Here we use gemini-pro modal for our use.
and use flask for web pages and routes.

How to use-->
1. first clone the repo.
2. get google api key from google gemini servies. link-> https://aistudio.google.com/app/apikey
3. make a .env file set google_api_key= your api key.
4. Install all dependencies by run the command -> pip install -r requirements.txt
5. run the project by run the command-> python app.py
6. check on your localhost.

Deployed Link-> https://agriculture-chatbot.onrender.com/
